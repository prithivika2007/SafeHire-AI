# SafeHire AI 🌿

**Career Re-entry Assistant for Women**  
Built for SheBuilds Chennai Hack — Code & Challenge 3.0  
SDG 5: Gender Equality

## What it does
An AI-powered platform that helps women return to the workforce after career breaks. Features include:
- Resume Gap Analyzer
- Personalized Career Roadmap
- Returnship Finder
- AI Interview Coach
- Confidence Booster Dashboard
- Emotional Readiness Check (unique feature)

## Stack
- Pure HTML / CSS / JavaScript (no framework)
- Claude AI API (claude-sonnet-4-6)

## Setup
1. Clone the repo
2. Add your Claude API key in `dashboard.html` (or via environment config)
3. Open `index.html` with Live Server in VS Code
